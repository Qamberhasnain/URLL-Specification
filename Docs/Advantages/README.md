# Advantages of the URLL format

URLL's main benefits over the typical `.*url` or `*.lnk` files is that it:

- [x] Is plain text
- [x] Is cross-platform
- [x] Can store multiple links, and doesn't require a strict syntax, so it is easy to learn.
